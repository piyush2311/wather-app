# Weather Application

A responsive weather application that displays current weather conditions and forecast for any location.

## Features

- Current weather conditions
- 5-day weather forecast
- Search by city name
- Responsive design for all devices
- Temperature unit toggle (Celsius/Fahrenheit)

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- [OpenWeatherMap API](https://openweathermap.org/api)

## Getting Started

### Prerequisites

- Web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- API key from OpenWeatherMap (sign up at [OpenWeatherMap](https://home.openweathermap.org/users/sign_up))

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/piyush2311/wather-app
   ```

2. Navigate to the project directory:
   ```bash
   cd weather-app2
   ```

3. Open `index.html` in your browser

## Usage

1. Enter a city name in the search bar
2. Press Enter or click the search button
3. View current weather and forecast

## Configuration

1. Get your API key from [OpenWeatherMap](https://home.openweathermap.org/api_keys)
2. Create a `.env` file in the root directory
3. Add your API key:
   ```
   API_KEY=your_api_key_here
   ```
